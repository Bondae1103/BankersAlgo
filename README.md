# Banker's Algorithm Offline Visualizer

This project is an **interactive offline simulation** of the Banker's Algorithm (deadlock avoidance). It demonstrates how safe sequences are determined and detects deadlock visually.

## Features
- Completely **offline** (no server required)
- **Process-specific boxes** showing Max and Allocation matrices
- **Available** resource vector
- **Animation panel** visualizing the algorithm step-by-step
- **Run**, **Pause**, **Step**, and **Reset** controls
- Displays whether the system is **safe or deadlocked** at the end

- `index.html` — main interface
- `css/style.css` — styling
- `js/app.js` — algorithm + animation logic
