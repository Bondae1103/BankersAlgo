# User Instructions

## How to Run
1. Extract the zip file.
2. Double-click **index.html** (works offline, no setup needed).

## How to Use
1. Choose the number of **Processes** and **Resources**.
2. Click **Generate Random** to populate matrices randomly, or **Load Preset Example** for a textbook case.
3. Review the **Available**, **Max**, and **Allocation** matrices (each process box is labeled P0, P1, etc.).
4. Click **Run Simulation** to execute the Banker's Algorithm.
5. Observe the animation and final result message:
   - ✅ All processes completed successfully → Safe sequence displayed.
   - ⚠️ Deadlock detected → Unsafe state message shown.

## Additional Controls
- **Step ▶ / ◀ Step** — move through recorded algorithm steps manually.
- **Reset** — clears the animation and result message.
- **Speed Slider** — controls animation playback speed.
- **Screenshot** — saves the current canvas view as an image.

## Notes
- Works best on Chrome, Edge, or Firefox.
- Supports up to 10 processes and 6 resource types.
